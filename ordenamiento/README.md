# Proyecto: Algoritmos de Ordenamiento

Este proyecto contiene implementaciones prácticas de los algoritmos de ordenamiento fundamentales estudiados en la Semana 4.

## Contenido de la Semana 4

1.  **Ordenamiento por Selección (Selection Sort):** Busca el elemento mínimo y lo coloca al inicio.
2.  **Ordenamiento por Inserción (Insertion Sort):** Inserta cada elemento en su posición correcta en una lista ya ordenada.
3.  **Ordenamiento Shell (Shell Sort):** Una versión optimizada del ordenamiento por inserción que utiliza "gaps" (espacios).

## Cómo ejecutar

1.  Abre la carpeta `ordenamiento` en VS Code.
2.  Ve a la carpeta `src`.
3.  Ejecuta el archivo `App.java`. Este abrirá un menú interactivo en la consola para elegir qué ejercicio probar.
