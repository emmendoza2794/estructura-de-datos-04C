# Proyecto: Arrays y Búsqueda

Este proyecto contiene ejemplos prácticos sobre el manejo de estructuras de datos lineales (Arreglos/Arrays) y algoritmos fundamentales de búsqueda y ordenamiento en Java.

## Contenido de la Semana 3

1.  **Arreglos Unidimensionales (Vectores):** Declaración, inicialización y recorrido.
2.  **Operaciones con Arreglos:** Cálculos estadísticos básicos (Suma, Promedio, Máximo, Mínimo).
3.  **Búsqueda Lineal:** Algoritmo secuencial de búsqueda.
4.  **Ordenamiento Burbuja (Bubble Sort):** Organización de datos.
5.  **Búsqueda Binaria:** Búsqueda eficiente en arreglos ordenados.

## Cómo ejecutar

1.  Abre la carpeta `arrays y busqueda` en VS Code.
2.  Ve a la carpeta `src`.
3.  Ejecuta el archivo `App.java`. Este abrirá un menú interactivo en la consola para elegir qué ejercicio probar.
